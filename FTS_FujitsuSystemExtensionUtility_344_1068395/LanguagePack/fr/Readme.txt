 Informations générales
=====================================================================

 L'utilitaire d'extension du système Fujitsu est un logiciel prenant  
 en charge diverses fonctions d'extension du système des ordinateurs 
 portables Fujitsu LIFEBOOK.

=====================================================================
 Installation de l'utilitaire d'extension du système Fujitsu 
=====================================================================

 - Pour pouvoir installer ce logiciel, vous devez ouvrir une session 
   Etitre d'administrateur. 
 - Avant d'installer l'utilitaire d'extension du système Fujitsu, 
   veuillez installer le pilote de périphérique FUJ02E3 (version 1.00 
   ou plus récente) séparément. Pour installer ce pilote, veuillez  
   consulter le fichier lisez-moi (readme) FUJ02E3.
 - L'exécution du programme Setup.exe permet d'installer l'utilitaire 
   d'extension du système Fujitsu. Une fois l'exécution lancée, une  
   boû‘e de dialogue apparaû‘ El’écran. Suivez les instructions 
   affichées dans la fenêtre pour procéder El'installation. 

=====================================================================
 Désinstallation de l'utilitaire d'extension du système Fujitsu 
=====================================================================

 - Pour Windows XP, Windows Vista ou Windows 7, connectez-vous en tant 
   qu'administrateur pour désinstaller le programme. 
 - Ne procédez pas Eune désinstallation lorsque plusieurs utilisateurs 
   sont connectés via la fonction « Changer d'utilisateur » de 
   Windows XP, Windows Vista ou Windows 7. La désinstallation ne peut 
   être faite que si un seul utilisateur est connectE

Procédure de désinstallation :

 1. Avant de commencer la désinstallation, fermez toutes les applications 
    ouvertes.
 2. Cliquez sur Démarrer -> Paramètres -> Panneau de configuration.
 3. Cliquez sur « Ajouter ou supprimer des programmes » si vous avez 
    Windows XP, ou sur « Désinstaller un programme » si vous avez 
    Windows Vista ou Windows 7. (Sous Programmes)
 4. Dans la liste, sélectionnez l'utilitaire d'extension du système 
    Fujitsu (Fujitsu System Extension Utility) et cliquez sur 
   [Modifier/Supprimer] si vous avez Windows XP ou sur [Désinstaller] 
    si vous avez Windows Vista ou Windows 7.
    Le système affiche une fenêtre de configuration de l'utilitaire 
    d'extension du système Fujitsu. 
 5. Fermez le Panneau de configuration, puis cliquez sur [OK]. La 
    désinstallation commence. 

 Une fois désinstallée, la fonction d'extension matérielle ne peut plus 
 être utilisée sur l'ordinateur portable de série LIFEBOOK. Pour utiliser 
 la fonction d'extension sur ce type d'ordinateur, vous devez 
 réinstaller l'utilitaire d'extension du système Fujitsu.

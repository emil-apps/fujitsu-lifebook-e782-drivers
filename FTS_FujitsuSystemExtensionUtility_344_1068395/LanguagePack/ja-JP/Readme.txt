■ 6.インストール手順

  1) コンピューターの管理者のアカウント、あるいは管理者または
     Administratorsグループのメンバーとしてログオンします。
  2) 画面左下の「スタート」ボタンをクリックし、
     「スタート」メニューを開きます。
  3) 次の手順にて本ソフトウェアが格納されているフォルダーへ移動します。

     - Windows 7 の場合
       1) 「スタート」メニューの「コンピューター」をクリックします。
          「コンピューター フォルダー」が表示されます。
       2) 「コンピューター フォルダー」から、本ソフトウェアが格納されて
          いるドライブ、フォルダーへ移動します。

     - Windows Vista の場合
       1) 「スタート」メニューの「コンピュータ」をクリックします。
          「コンピュータ フォルダ」が表示されます。
       2) 「コンピュータ フォルダ」から、本ソフトウェアが格納されている
          ドライブ、フォルダーへ移動します。

     - Windows XP の場合
       1) 「スタート」メニューの「マイ コンピュータ」をクリックします。
          「マイ コンピュータ」が表示されます。
       2) 「マイ コンピュータ」から、本ソフトウェアが格納されている
          ドライブ、フォルダーへ移動します。

  4) 「Setup」または「Setup.exe」をクリックします。
  5) 以降は表示された画面に従って操作してください。

  以上でインストールは終了です。

------------------------------------------------------------------------
■ 7.インストール確認手順

  1) 画面左下の「スタート」ボタンをクリックし、「スタート」メニューを開
     きます。
  2) 「スタート」メニューから「コントロール パネル」をクリックします。
  3) 下記の手順にて、本ソフトウェアがインストールされていることを確認し
     ます。

     - Windows 7 の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ コントロール パネル」の表示方法を「大きいアイコン」または
             「小さいアイコン」にし、全てのアイコンが表示される状態にし
             ている場合は、「プログラムと機能」アイコンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「富士通拡張機能ユーティリティ」が表示されていることを確認しま
          す。

     - Windows Vista の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ 「コントロールパネル」を「クラシック表示」で全てのアイコン
             が表示される状態にしている場合は、「プログラムと機能」アイ
             コンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「富士通拡張機能ユーティリティ」が表示されていることを確認し
          ます。

     - Windows XP の場合
       1) 「プログラムの追加と削除」アイコンをクリックします。
       2) 表示された一覧の中に「富士通拡張機能ユーティリティ」が表示さ
          れていることを確認します。

■ 8.アンインストール手順

  1) 画面左下の「スタート」ボタンをクリックし、「スタート」メニューを開
     きます。
  2) 「スタート」メニューから「コントロール パネル」をクリックします。
  3) 下記の手順にて、アンインストールを行います。

     - Windows 7 の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ コントロール パネル」の表示方法を「大きいアイコン」または
             「小さいアイコン」にし、全てのアイコンが表示される状態にし
             ている場合は、「プログラムと機能」アイコンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「富士通拡張機能ユーティリティ」を選択し、
          「アンインストール」をクリックします。

     - Windows Vista の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ 「コントロールパネル」を「クラシック表示」で全てのアイコン
             が表示される状態にしている場合は、「プログラムと機能」アイ
             コンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「富士通拡張機能ユーティリティ」を選択し、
          「アンインストール」をクリックします。

     - Windows XP の場合
       1) 「プログラムの追加と削除」アイコンをクリックします。
       2) 表示された一覧の中から「富士通拡張機能ユーティリティ」を選択
          し、「削除」をクリックします。

  4) 以降は表示された画面に従って操作してください。

  以上でアンインストールは終了です。

  ※ アンインストール時の留意点

   - 「ユーザーの切り替え」機能により、複数のユーザーでログオンした状態
     で、アンインストールを行わないでください。アンインストールを行うユ
     ーザーのみがログオンした状態で、アンインストールを行ってください。

   - アンインストールする場合は、コンピューターの管理者であるユーザーで
     行ってください。

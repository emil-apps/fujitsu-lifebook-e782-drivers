Help for Fujitsu Display Manager

 

Brief

Fujitsu Display Manager (hereinafter referred to as "this Software") is a utility that allows you to easily switch screen display settings for your computer.
Using this Software, you can save certain combinations of screen-displays as preset settings and restore your saved setting as needed. Fujitsu Display Manager is very helpful in the following types of situations:

  • When you're making a presentation by connecting a projector to your system.
  • When you're switching between multi-display and single-display modes.
  • When you're playing a DVD on a TV that is connected to your system.

How to use Fujitsu Display Manager

Starting up Fujitsu Display Manager


There are three different ways to start Fujitsu Display Manager:

  • Click [Start] -> [All Programs] -> [Fujitsu Display Manager].
  • Push Fn + F10 and select a menu item to start Fujitsu Display Manager.
  • Double-click the Fujitsu Display Manager icon on the task bar. (To display the icon, follow the procedure below).

Use the following procedure to place the Fujitsu Display Manager icon on the task bar.

  1. Start up Fujitsu Display Manager : Click [Start] -> [All Programs] -> [Fujitsu Display Manager] to open [Fujitsu Display Manager].
  2. Select [Setting] in the displayed [Fujitsu Display Manager] window to show the menu.
  3. Select the [Show icon on the taskbar] menu item.

Using the icon on the taskbar


By double-clicking the Fujitsu Display Manager icon on the taskbar, or right-clicking the icon and selecting [Setting...], you can view the main window for this Software.
Right-clicking the icon and selecting the display profile, you can apply the selected display profile.

How to use Fujitsu Display Manager


When the Fujitsu Display Manager starts, a main window appears. Using the arrow keys, select a display profile or the function you would like to use, then press [Enter].

Selection

Description

Profiles

View your display profile list.

Save current display setting as a display profile

Add the current screen-display setting to your [Profile] list.
If selected, a dialog box appears in which to enter the title of the new display profile. Enter the title of the new display profile, and click [OK].
Note : The maximum number of display profile is 16.

Setting

If selected, a pop-up menu appears in which to perform following features :

  • Up/Down : Use the [Up]/[Down] menu item to move a selected display profile on list upward or downward re-position items in your Profile list.
  • Delete : Select [Delete] to remove the selected display profile from the Profile list.
  • Edit : Select [Edit] to open the dialog box for editing existing display profile.
  • Create new display profile : Select [Create new display profile] to create a new display profile to add to Profile list.
    To make a new displa profile, you must first have the display device you wish to use connected to the computer, then select [Create new display profile].
    Note : The maximum number of display profile is 16.
  • Show icon on the taskbar : If checked, the Fujitsu Display Manager icon will appear on the taskbar. If unchecked, the icon will not appear on the taskbar.
  • Restore factory default display setting : Select [Restore factory default display setting] to apply the factory default display setting.

 

Using the dialog box for editing display profile


Selection

Description

Title of the display profile

Used for editing the title of the display profile.

Title of the display profile

Used for editing the title of the display profile.

Display mode

Use this field to select display mode from the drop-down list. Selections include the following:

  • Single display mode : Select to display on a single display connected to the computer.
  • Clone display mode : Select to display the same image on two individual displays connected to the computer.
  • Multi-display mode : Select to display one desktop on two combined displays. The display which shows the start menu is called the [Primary display]; the other is called the [Secondary display].

Notes :

  • When default registerd display profile is selected on the [Main window], you cannot make changes to this field.
  • You can select [Clone display] or [Multi-display] only when two displays are connected to the computer.
  • You can not select the Second Display as the [Primary display] and the [Secondary display] of the Clone display mode.

Primary display/Secondary display

The Primary and Secondary display fields allow you to select displays from the drop-down lists.

Position of secondary display/Alignment of secondary display

When [Multi-display mode] is selected, you can specify the position of the secondary display.

Settings

You can view selectable combinations of resolutions and refresh rates so as to select the combination you wish to apply.
Note : The combination of selectable resolutions and refresh rates depends on the capabilities of your graphic accelerator and the computer display.

 

Points to consider

  • While you are playing video on Windows Media Center or another application, do not start up Fujitsu Display Manager.
  • Display switching may change the position or size of icons or windows.
  • If a display in use does not support the resolution or refresh rate registered to a display profile or is different from the one used when registered to display profile, the display profile will not work. You must use the display profile which applies to the resolution or refresh rate supported by the display being used.

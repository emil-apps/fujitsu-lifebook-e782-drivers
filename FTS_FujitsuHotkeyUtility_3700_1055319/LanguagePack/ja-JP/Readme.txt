    り画面の明るさや、音声ボリューム、内蔵ポインティングデバイスの状態
    の変更を行なっても、インジケーターが画面上に表示されない場合があり

      - ログオン画面やロック解除画面

    ることがあります。
    その場合は、システムの再起動を行ってください。


  - 音量ボリュームの値は100段階になっていますが、インジケーター表示は
    26段階で行っています。
    そのため、１回の操作に対してインジケーター表示のメモリが変わらない
    ことがあります。

■ 6.インストール手順

  1) Windows にアカウントの種類が「コンピューターの管理者」である
     ユーザーのみでログオンします。複数のユーザーがログオンした状態で
     は、インストールできませんので、ご注意ください。
  2) 画面左下の「スタート」ボタンをクリックし、「スタート」メニューを
     開きます。
  3) 次の手順にて本ソフトウェアが格納されているフォルダーへ移動します。

     - Windows 7 の場合
       1) 「スタート」メニューの「コンピューター」をクリックします。
          「コンピューター フォルダー」が表示されます。
       2) 「コンピューター フォルダー」から、本ソフトウェアが格納されて
          いるドライブ、フォルダーへ移動します。

     - Windows Vista の場合
       1) 「スタート」メニューの「コンピュータ」をクリックします。
          「コンピュータ フォルダ」が表示されます。
       2) 「コンピュータ フォルダ」から、本ソフトウェアが格納されている
          ドライブ、フォルダーへ移動します。

  4) 「Setup」または「Setup.exe」をクリックします。
  5) 以降は表示された画面に従って操作してください。

  以上でインストールは終了です。

------------------------------------------------------------------------
■ 7.インストール確認手順

  1) 画面左下の「スタート」ボタンをクリックし、「スタート」メニューを開
     きます。
  2) 「スタート」メニューから「コントロール パネル」をクリックします。
  3) 下記の手順にて、本ソフトウェアがインストールされていることを確認し
     ます。

     - Windows 7 の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ コントロール パネル」の表示方法を「大きいアイコン」または
             「小さいアイコン」にし、全てのアイコンが表示される状態にし
             ている場合は、「プログラムと機能」アイコンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「IndicatorUtility」が表示されていることを確認します。

     - Windows Vista の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ 「コントロールパネル」を「クラシック表示」で全てのアイコン
             が表示される状態にしている場合は、「プログラムと機能」アイ
             コンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「IndicatorUtility」が表示されていることを確認します。

------------------------------------------------------------------------
■ 8.アンインストール手順

  1) 画面左下の「スタート」ボタンをクリックし、「スタート」メニューを開
     きます。
  2) 「スタート」メニューから「コントロール パネル」をクリックします。
  3) 下記の手順にて、アンインストールを行います。

     - Windows 7 の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ コントロール パネル」の表示方法を「大きいアイコン」または
             「小さいアイコン」にし、全てのアイコンが表示される状態にし
             ている場合は、「プログラムと機能」アイコンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「IndicatorUtility」を選択し、「アンインストール」をクリックし
          ます。

     - Windows Vista の場合
       1) 「プログラムのアンインストール」をクリックします。
          ※ 「コントロールパネル」を「クラシック表示」で全てのアイコン
             が表示される状態にしている場合は、「プログラムと機能」アイ
             コンをクリックします。
       2) 「プログラムのアンインストールまたは変更」が開いたら、
          「IndicatorUtility」を選択し、「アンインストール」をクリックし
          ます。

  4) 以降は表示された画面に従って操作してください。

  以上でアンインストールは終了です。

  ※ アンインストール時の留意点

   - 「ユーザーの切り替え」機能により、複数のユーザーでログオンした状
     態で、アンインストールを行わないでください。アンインストールを行
     うユーザーのみがログオンした状態で、アンインストールを行ってくだ
     さい。

   - アンインストールする場合は、コンピューターの管理者であるユーザー
     で行ってください。

------------------------------------------------------------------------
■ 9.ご使用方法

  IndicatorUtility は、パソコンの画面上にインジケーターを表示するユー
  ティリティです。
  キーボード、リモコン、本体上のボタンなどから、画面の明るさや、音声ボ
  リューム、内蔵ポインティングデバイスの状態が変更された時、インジケー
  ターを表示します。

  表示されるインジケーターは、お使いの機種や BIOS の設定によって異なり
  ます。

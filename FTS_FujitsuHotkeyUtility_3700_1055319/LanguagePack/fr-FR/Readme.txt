===================================================================
 Récapitulatif de l'utilitaire Hotkey
===================================================================

 L'utilitaire Hotkey affiche des voyants El'écran correspondant 
 aux raccourcis clavier (Fn+F3, F4, F6, F7, F8, F9*).

 * Les raccourcis de clavier disponibles dépendent du modèle de 
   l'ordinateur et des paramètres de votre BIOS.

===================================================================
 Installation de l'utilitaire Hotkey
===================================================================

 1. Pour installer l'utilitaire Hotkey, vous devez vous connecter 
    en tant qu'administrateur (mode Administrateur) avec un 
    compte d'administrateur.

 2. Avant d'utiliser l'utilitaire Hotkey, installez le pilote de 
    périphérique FUJ02B1 séparément. Pour les instructions 
    d'installation du pilote de périphérique FUJ02B1 (version 1.23 
    ou plus récente), consultez les instructions du périphérique
    FUJ02B1.

 3. Avant d'installer l'utilitaire Hotkey, fermez toutes les 
    applications ouvertes.

 4. Pour installer l'utilitaire Hotkey, exécutez le fichier 
    Setup.exe, puis suivez les instructions s'affichant dans la 
    fenêtre de configuration de l'utilitaire Hotkey. 

 5. Pour réinstaller l'utilitaire Hotkey, vous devez d'abord 
    désinstaller l'utilitaire Hotkey actuellement en place. 

===================================================================
 Désinstallation de l'utilitaire Hotkey (suppression du programme installE
===================================================================

 1. Pour désinstaller l'utilitaire Hotkey, vous devez vous 
    connecter en tant qu'administrateur (mode Administrateur) avec 
    un compte d'administrateur. En outre, pour désinstaller cet 
    utilitaire, aucun autre utilisateur ne doit être connectE
    au système.

 2. Avant de commencer la désinstallation, fermez toutes les 
    applications ouvertes.

 3. Pour désinstaller l'utilitaire, exécutez les opérations 
    suivantes :
    a) Cliquez sur Démarrer -> Panneau de configuration.
    b) Dans le Panneau de configuration, cliquez sur « Désinstaller 
       un programme ».
    c) Dans la liste affichée, choisissez « Utilitaire Hotkey ». 
       Cliquez sur le bouton [Désinstaller]. L'utilitaire Hotkey 
       est alors effacE

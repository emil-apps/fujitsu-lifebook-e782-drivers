===================================================================
 Panoramica
===================================================================

 Il software Hotkey Utility consente di visualizzare sullo schermo 
 la barra di stato  dei tasti di scelta rapida (Hotkey) del 
 Notebook. (per es. Fn+F3, F4, F6, F7, F8, F9*).

 * Il numero di combinazioni di tasti e di funzioni a disposizione 
   dipende dal modello e dalle impostazioni del BIOS.

Installazione di Hotkey Utility
===================================================================

 1. Per installare il software Hotkey Utility ・necessario 
    effettuare il login come amministratore.
	
 2. Prima di utilizzare il software Utility Hotkey, installare 
    eventualmente i driver FUJ2B1. 

 3. Chiudere tutte le applicazioni aperte.

 4. Fare clic sul file Setup.exe per installare il software 
    Hotkey Utility e seguire le istruzioni. 

 5. Per reinstallare il software Hotkey Utility ・necessario per 
    prima cosa disinstallarlo.

===================================================================
Disinstallazione di Hotkey Utility
===================================================================

 1. Per disinstallare Hotkey Utility ・necessario effettuare il 
    login come amministratore.

 2. Chiudere tutte le applicazioni aperte.

 3. Eseguire passo a passo le istruzioni di seguito descritte:
	a) Selezionare nel menu Start la voce Pannello di controllo
	b) Fare clic su Disinstalla un programma.
	c) Selezionare dall'elenco la voce Fujitsu Hotkey Utility 
           e far clic su Disinstalla.

===================================================================
Note
===================================================================

 - Dopo aver modificato le Impostazioni schermo o quando
   lo Screen saver ・attivo, pu・non comparire alcuna 
   barra di stato. I tasti di scelta rapida funzionano comunque 
   normalmente.

 - Se avete finalizzato l'accesso con i privilegi di amministratore, 
   potrebbe non comparire    l'icona del volume. I tasti di scelta 
   rapida continuano comunque a funzionare normalmente.

 - A volte una finestra pu・oscurare 
   la visualizzazione. In tal caso, riavviare il sistema.
 
 - L'icona del volume presenta 26 diversi livelli, anche se 
   effettivamente il Notebook ne supporta 100. Per questo motivo 
   l'icona potrebbe restare invariata dopo la pressione di un tasto.

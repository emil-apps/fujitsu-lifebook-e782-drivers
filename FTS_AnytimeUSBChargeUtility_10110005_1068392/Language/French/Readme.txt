  Vous pouvez modifier les  paramètres du « Chargeur USB Anytime » pour  
  permettre à l'ordinateur de charger des périphériques USB sans être mis en 
  marche grâce à l'utilisation de « l'Utilitaire du Chargeur USB » 
  (« le Logiciel »).

  Ce manuel fournit des informations détaillées sur « l'Utilitaire du Chargeur 
  USB ». Le contenu de ce manuel est comme suit :

  3.Configurations système nécessaires

  8.Comment utiliser l'utilitaire

  3. Configurations système nécessaires

  Pour utiliser « l'utilitaire de charge USB Anytime », votre système doit 
  répondre aux exigences suivantes.

  
  Systèmes d’exploitation compatibles :
    -  Windows(R) 7 Édition Familiale de Base 64 bit
    -  Windows(R) 7 Édition Familiale Supérieure 64 bit
    -  Windows(R) 7 Édition Professionnelle 64 bit
    -  Windows(R) 7 Édition Entreprise 64 bit
    -  Windows(R) 7 Édition Ultime 64 bit


  Espace disque dur : 
    Plus de 5 Mo d'espace disponible

      REMARQUE : N'exécutez le logiciel que sur un ordinateur que vous avez 
                 acheté.

  5.Installation

  1) Mettez l'ordinateur en marche et connectez-vous en tant qu'administrateur 
     ou membre du groupe administratif.

  2) Cliquez sur le bouton [Démarrer] et ouvrez le menu [Démarrer].

  3) Suivez les instructions qui s'affichent selon votre système d'exploitation :

     Dans Windows(R) 7 :

     1. Cliquez sur [Ordinateur] dans le menu [Démarrer] et vous verrez 
        [Ordinateur].

     2. A partir de [Ordinateur], naviguez jusqu'au dossier ou au disque dans 
        lequel le logiciel est enregistré.


  4) Cliquez sur le fichier « Setup » ou « Setup.exe »

  5) Suivez les instructions figurant sur la fenêtre qui s'affiche.

  L'installation est terminée.

-------------------------------------------------------------------------------

  6.Comment confirmer l'installation

  1) Cliquez sur le bouton [Démarer] pour ouvrir le menu [Démarrer].

  2) Sélectionnez [Tous les programmes] dans le menu [Démarrer].

  3) Sélectionnez [Chargeur USB Anytime ] -> [Paramètres].

  4) Assurez-vous que la fenêtre de « l'Utilitaire du chargeur USB Anytime » 
     s'affiche.

-------------------------------------------------------------------------------

  7.Désinstallation

  1) Cliquez sur le bouton [Démarer] pour ouvrir le menu [Démarrer].

  2) Cliquez sur [Panneau de configuration] dans le menu[Démarrer].

  3) Effectuez les étapes qui s'affichent selon votre système d'exploitation :

     Dans Windows(R) 7 :

     1. Cliquez sur [Désinstaller le programme].

        - Si vous avez sélectionné [Grande Icône] ou [Petite] pour afficher 
          le [Panneau de configuration], cliquez sur [Programmes et 
          Fonctionnalités].

     2. Sélectionnez « Utilitaire du Chargeur USB Anytime » dans la fenêtre 
        [Désinstaller un programme] ensuite, cliquez sur [Désinstaller et 
        modifier].

     3. Suivez les instructions figurant sur la fenêtre qui s'affichent 
        à l'écran.


  La désinstallation est terminée.

  REMARQUES sur la désinstallation :

     - Ne procédez pas à une désinstallation du logiciel lorsque plusieurs 
       utilisateurs sont connectés sur l'ordinateur.
       Il ne doit y avoir qu'une seule personne connectée sur l'ordinateur 
       pendant la désinstallation.

     - Il faut être connecté en tant qu'utilisateur pour pouvoir effectuer 
       la désinstallation du logiciel.

-------------------------------------------------------------------------------

  8.Comment utiliser l'utilitaire

  Veuillez consulter l'Aide ci-jointe pour voir les consignes d'utilisation 
  de « l'Utilitaire du Chargeur USB Anytime ». Pour acceder à l'Aide, exécuter 
  les étapes suivantes :

  1) Cliquez sur le bouton [Démarrer] pour ouvrir le menu [Démarrer].

  2) Cliquez sur [Tous les programmes] dans le menu [Démarrer].

  3) Sélectionnez [Chargeur USB Anytime] -> [Aide].
